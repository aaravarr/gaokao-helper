# ⭐ 志愿星 — 高考志愿填报助手

**考多少分，上什么大学？**

输入省份和分数，30秒获取 冲一冲 / 稳一稳 / 保一保 志愿推荐。

🔗 **直接使用：https://aaravarr.github.io/gaokao-helper/**

## ✨ 特点

- **完全免费** — 无注册、无广告、无会员、无套路
- **真实数据** — 2022-2025年百度高考录取分数线，非模拟数据
- **覆盖面广** — 899所高校（985/211/双一流），全国31省
- **智能推荐** — 基于分数差的 冲/稳/保 三级匹配算法
- **筛选系统** — 按层次（985/211/双一流）、区域→城市、11种学校类型筛选
- **趋势分析** — 历年分数线折线图 + 趋势箭头（↑→↓）
- **跨省参考** — 省份数据不足时自动计算全国平均，标注"参考"
- **位次推荐** — 输入位次时使用更精准的位次匹配算法
- **秒出结果** — 纯前端，无需等待，输入即出
- **一键分享** — 生成带参数链接，发给同学/家长直接查看

## 🎯 使用方法

1. 选择你的省份
2. 输入高考分数（可选填位次）
3. 选择科类（新高考省份自动适配）
4. 点击"查询推荐"
5. 查看 冲/稳/保 三档推荐
6. 可使用筛选器缩小范围（层次/区域/城市/学校类型）

## 🏗️ 技术实现

纯静态前端应用，零服务器成本：

- **前端**: HTML + CSS + JavaScript（单文件，无框架依赖）
- **数据**: 紧凑JSON格式（7.0MB），gzip传输约1.7MB
- **部署**: GitHub Pages（免费静态托管）
- **算法**: 分数差匹配（冲=+5~+25, 稳=-5~+5, 保=-30~-5）+ 位次比对

### 数据来源
百度高考 API（gaokao.baidu.com），覆盖全国 899 所高校的录取数据。

### 新高考适配
- 3+3 省份（浙江、上海、北京、天津、山东、海南）：不分文理，科目为"综合"
- 3+1+2 省份（21省）：物理类/历史类
- 传统省份（新疆、西藏）：理科/文科
- 2025年新改革省份（四川、河南、山西、云南、陕西、青海、宁夏、内蒙古）：已支持
- 海南 900 分制已支持

## 🔍 数据发现了什么

分析 260,955 条真实录取数据后，我们发现了一些反直觉的事实：

1. **211 标签会骗人** — 70/77 所 211 高校在至少一个省超过了最易考的 985（北邮 612 > 西农 572）
2. **城市溢价值 45 分** — 北京高校平均 615 分，兰州 562 分，差 53 分，质量差距远没这么大
3. **211 和双一流只差 1 分** — 562 vs 561，标签不同，分数几乎一样
4. **隐藏捡漏校** — 延边大学 501 分、石河子大学 508 分，都是正经 211
