# ⭐ 志愿星 — 高考志愿填报助手

**考多少分，上什么大学？**

输入省份和分数，30秒获取 冲一冲 / 稳一稳 / 保一保 志愿推荐。

🔗 **直接使用：https://aaravarr.github.io/gaokao-helper/**

## ✨ 特点

- **完全免费** — 无注册、无广告、无会员、无套路
- **真实数据** — 2022-2024年百度高考录取分数线，非模拟数据
- **覆盖面广** — 200+所高校（985/211/双一流），全国31省
- **智能推荐** — 基于分数差的 冲/稳/保 三级匹配算法
- **跨省参考** — 省份数据不足时自动计算全国平均，标注"参考"
- **秒出结果** — 纯前端，无需等待，输入即出
- **一键分享** — 生成带参数链接，发给同学/家长直接查看

## 🎯 使用方法

1. 选择你的省份
2. 输入高考分数
3. 选择科类（新高考省份自动适配）
4. 点击"查询推荐"
5. 查看 冲/稳/保 三档推荐

## 🏗️ 技术实现

纯静态前端应用，零服务器成本：

- **前端**: HTML + CSS + JavaScript（单文件，无框架依赖）
- **数据**: JSON 格式内嵌（2.8MB），加载后本地匹配
- **部署**: GitHub Pages（免费静态托管）
- **算法**: 分数差匹配（冲=+5~+25, 稳=-5~+5, 保=-30~-5）

### 数据来源
百度高考 API（gaokao.baidu.com），覆盖全国 591 所高校的录取数据。

### 新高考适配
- 3+3 省份（浙江、上海、北京、天津、山东、海南）：不分文理
- 3+1+2 省份（21省）：物理类/历史类
- 传统省份（新疆、西藏）：理科/文科
- 海南 900 分制已支持

